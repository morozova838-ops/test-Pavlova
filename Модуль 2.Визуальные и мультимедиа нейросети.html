Задание 2.1. Создание комплекта дидактических карточек

25 баллов

Цель: Научиться генерировать точные, детализированные и эстетически выдержанные изображения для уроков без нарушения авторских прав.

Что нужно сделать:

  • Откройте мобильное приложение или веб-версию нейросети Шедеврум (или ЯндексART).
  • Используйте предложенный ниже шаблон промпта, адаптировав его под тему своего предмета. Чтобы серия картинок выглядела цельной, не меняйте параметры стиля в конце запроса.
  • Сгенерируйте минимум 3 карточки (например: 3 архитектурных стиля, 3 вида клеток, 3 физических явления).
  • Скачайте изображения и загрузите их в СДО в формате архива или единого PDF-документа.
Шаблон промпта для генерации картинок

Иллюстрация для школьного учебника. Изолированный объект: [впишите конкретный объект, например: микроскоп / замок XIV века / молекула воды] на чистом однотонном светлом фоне. Высокая детализация, мягкий свет, академический стиль, понятный для детей, 3D рендеринг, минимализм, профессиональный графический дизайн.

💡 Полезный совет методиста: Избегайте в промптах абстрактных слов («красивый», «невероятный»). Нейросети гораздо лучше понимают конкретные технические параметры: «светлый фон», «высокая детализация», «минимализм».

Задание 2.2. Проектирование обложки учебного курса или презентации

25 баллов

Цель: Научиться использовать метафорическое мышление при работе с генеративным дизайном для создания вовлекающего визуального контента.

Что нужно сделать:

  • Придумайте яркую визуальную метафору для вашего предмета или методического проекта.
  • Сформулируйте запрос на основе шаблона и добейтесь от нейросети качественного графического паттерна или главного изображения для стартового слайда презентации.
  • Прикрепите в СДО получившуюся обложку и напишите краткое пояснение (1-2 предложения), почему именно этот образ вы выбрали для учеников.
Шаблон промпта для обложки / презентации

Современный плоский векторный логотип и концепт обложки для образовательного курса по [укажите дисциплину, например: геометрии / литературе]. Визуальный образ: [опишите метафору, например: книга, из которой вырастает дерево / геометрические фигуры в виде футуристического города]. Стиль неоморфизм, яркие неоновые линии, глубокий синий и фиолетовый цвета, цифровое искусство, векторная графика, без текста.

Шаблоны разработаны с опорой на технические требования отечественных платформ ИИ. Вся поддержка по генерации артов — в нашем Telegram-канале.
